# ECHO PROTOCOL v0.7 - CHANGELOG

## 🚀 WHAT'S NEW

### ⏱️ REAL-TIME SHIFT MANAGEMENT
- **The Problem:** Unlimited time to explore, lost tension
- **The Solution:** Ticking clock that counts down every 5 seconds
- **Features:**
  - Game starts at 02:13
  - Normal: 12 hours (720 min)
  - Hard: 6 hours (360 min)
  - Speedrun: 6 hours race mode
  - Clock display top-right: `02:13 / 720MIN`
  - Color changes: Green → Yellow → Red as time runs out
  - GAME OVER when time expires

**Impact:** ⭐⭐⭐⭐⭐ Completely changes pacing


### 🔗 CLUE COMBINATION SYSTEM
- **The Problem:** Clues felt disconnected, no way to combine insights
- **The Solution:** Automatic detection when 2+ clues connect
- **Features:**
  - 20+ predefined combinations
  - Examples:
    - Foto + Fiber → Harbor scene unlocked
    - Band + Mirror text → Secret revelation
    - Last Signal + Location → New lead
  - Auto-trigger when collected
  - Unique narrative scenes
  - Extra rewards (Insight +2, Score +25)

**Impact:** ⭐⭐⭐⭐ Makes you think about connections


### 🎯 CONSEQUENCE PREVIEW
- **The Problem:** Choices felt blind, no way to predict outcomes
- **The Solution:** Hover over choice to see stat changes
- **Features:**
  - Shows: Shift ±1, Stress ±2, Trust +1, etc.
  - Color coded: Green (good), Red (bad), Blue (neutral)
  - Works on all 50+ choices
  - No spoilers - just mechanics preview

**Impact:** ⭐⭐⭐ Makes decisions more informed


### 📱 MESSAGE SYSTEM
- **The Problem:** One-way narrative, no communication
- **The Solution:** Dynamic message queue with notifications
- **Features:**
  - Radio intercepts from dispatch
  - Text messages from Jonas
  - Mira signals (mysterious)
  - System alerts for combo triggers
  - Message indicator: `📨 3 NEW`
  - Messages appear based on Insight level
  - Persistent unread counter

**Impact:** ⭐⭐⭐⭐ Makes world feel alive


### 🏆 RANK PROGRESS SYSTEM
- **The Problem:** No progression feeling, flat progression
- **The Solution:** 7-tier rank system with multipliers
- **Features:**
  - Ranks: Cadet → Officer → Detective → Senior → Expert → Master → Architect
  - Each decision: +10 points
  - Each combo: +50 points
  - Rank display top-right always visible
  - Multipliers for future score calculations
  - Prestige mechanic across playthroughs

**Impact:** ⭐⭐⭐⭐ Gives long-term goals


### 🎖️ ACHIEVEMENT SYSTEM
- **The Problem:** No goal tracking beyond story
- **The Solution:** 40+ achievement badges
- **Features:**
  - Categories: Gameplay, Story, Challenge, Speedrun
  - Examples:
    - "First Step" (make 1 decision)
    - "Pattern Seeker" (complete 3 combos)
    - "Speed Demon" (finish in < 150 min)
    - "Deep Archive" (explore all 16 districts)
    - "Deep Diver" (reach max insight)
  - Auto-unlock on trigger
  - Achievement tab shows all progress
  - Reward score on unlock

**Impact:** ⭐⭐⭐ Adds replayability


### 🎮 QUALITY OF LIFE IMPROVEMENTS
- **The Problem:** Save/load tedious, navigation slow
- **The Solution:** Keyboard shortcuts & quick access
- **Features:**
  - **F1-F8:** Quick load any slot (instant)
  - **Ctrl+S:** Quick save to active slot
  - **Ctrl+L:** Quick load from selected slot
  - Slot selector shows description
  - No more clicking through menus

**Impact:** ⭐⭐⭐⭐ Saves ~5 min per playthrough


### 🔥 HARD MODE & SPEEDRUN
- **The Problem:** Only one difficulty, no challenge modes
- **The Solution:** Mode selector at startup
- **Features:**
  - **Normal:** 720 minutes, baseline difficulty
  - **Hard:** 360 minutes, 2x time cost per choice
  - **Speedrun:** 360 minutes, 1x cost, race mode
  - Mode selection dialog on game start
  - Different achievements per mode
  - Locked saves (can't change mid-game)

**Impact:** ⭐⭐⭐⭐⭐ Massive replayability


### ✨ NEW STORY BRANCHES
- **The Problem:** Linear story, limited replay value
- **The Solution:** 15+ new scenes unlocked via combos
- **Features:**
  - Harbor Convergence (via photo+fiber combo)
  - Mirror Revelation (via band+mirror combo)
  - 13+ more combo-triggered scenes
  - Alternative Jonas confrontation
  - Secret archive paths
  - Multiple endings possible

**Impact:** ⭐⭐⭐⭐ Doubles replayability


### 🎨 ENHANCED UI/UX
- **The Problem:** Static UI, hard to parse info
- **The Solution:** Animated, color-coded, responsive UI
- **Features:**
  - Smooth transitions between scenes
  - Animated rain effect in background
  - Better color contrast (WCAG AA compliant)
  - Rank display always visible
  - Time display with color warning
  - Message indicator prominent
  - Clock position (top-right) always in view
  - Better meter bars (more visual)

**Impact:** ⭐⭐⭐ Makes game feel polished

### First Time Player
```
1. Start NORMAL mode (720 min)
2. Take your time reading narrative
3. Explore all 3 major choices
4. Try to get at least 2 clue combos
5. See if you can reach Jonas' ending

```

```
1. Start SPEEDRUN mode (360 min, 1x cost)
2. Know optimal choice path
3. Skip non-critical clues
4. Rush to ending
```
