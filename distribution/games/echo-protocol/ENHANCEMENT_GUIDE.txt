========================================

Enabled after completing game once.

Changes:
- Only 360 minutes (6 hours) instead of 720
- Stress increases +50% on wrong choices
- Trust penalties doubled
- Can't save/load during playthrough
- One save slot per Hard Mode run
- Wrong decisions have immediate consequences
- Some choices permanently locked out
- Archives require 2+ Insight to access
- Endings much harder to achieve

Rewards:
- +100% score multiplier
- Special "Hardened" badge on file
- Unlock "Architect's Regret" secret ending
- Bonus achievements

- Bronze: < 200 min
- Silver: < 120 min

where you become the next Architect.
