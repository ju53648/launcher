# Echo Protocol v0.7 - Enhancement Manifest

**Release Date:** April 29, 2026

## 🎮 GAMEPLAY REVOLUTION

### ⏱️ Real-Time Shift Management (NEW)
- **12-minute turns** with actual time pressure
- Each scene takes 1-5 minutes  
- Time remaining displayed at all times
- **3 modes:** Normal (720min) | Hard (360min) | Speedrun (360min)
- Bonus score for **speed clears**
- Color-coded urgency (green→yellow→red)

**Impact:** Game goes from unlimited exploration to tense time-management puzzle.

---

### 🔗 Clue Combination System (NEW)
- **Combine 2+ clues** for instant story revelations
- 20+ pre-designed combos (Photo + Fiber = Harbor Link)
- Unlocks **hidden scenes** and new story branches
- Rewards: **+25 Score, +2 Insight**
- Example combos:
  - "Photo #12" + "Faserprofil" → **Hafen-Verbindung** scene unlocks
  - "Band 08" + "Spiegeltext" → **Mirror Revelation** early access
  - "Jonas' Badge" + "Badge Abweichung" → **Jonas-Doubt** confrontation

**Impact:** Doubles replayability. Encourages pattern-finding. Rewards exploration.

---

### 🎯 Consequence Preview System (NEW)
- **Hover over choices** to see exact effects:
  - ⬆️ Shift delta (+1)
  - ❤️ Stress delta (+2)
  - 💭 Trust delta (-1)
  - ⏱️ Time cost (-3 min)
- Visual indicators:
  - 🟢 Green = Safe/Optimal path
  - 🟠 Orange = Risky but valuable
  - 🔴 Red = Locked/Bad path
  - 💡 Blue = Combo opportunity

**Impact:** Players understand consequences before committing. Less frustration, more strategy.

---

### 📱 Incoming Message System (NEW)
- **Real-time notifications** during play
- 4 message types:
  - 📻 Radio intercepts (from Dispatch)
  - 💬 Text messages (from Jonas)
  - ⚠️ System alerts (discoveries)
  - 🔴 Mira's cryptic signals (late-game)
- Messages can contain hints, warnings, or story progression
- Collect all messages for achievement
- Example:
  - [02:45] Jonas: "Found something in the archive. Don't go alone."
  - [03:00] System: "CLUE COMBO DISCOVERED"
  - [05:30] Mira: "The tower is listening. Don't trust the intercom."

**Impact:** Game feels alive. Immersion skyrockets. Tension builds gradually.
