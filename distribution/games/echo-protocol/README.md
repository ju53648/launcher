# Echo Protocol

Echo Protocol is a native desktop mystery game for Lumorix Launcher.

## Launcher target

The launcher starts:

`EchoProtocol.exe`

This game is distributed as a self-contained app and no longer depends on a browser or local web page.

## Local run

Run `EchoProtocol.exe`.

## Gameplay

- Story scenes with decisions
- 8 native save slots
- Reality shift text changes and locked late-game routes
- Expanded case file, leads, status tracking and timeline
- Multiple endings, containment paths and loop routes
- Deep Archive mega mode with 16 districts, 384 archive cases and 1,536 generated investigation layers
- Score, rank and achievement tracking for long challenge runs
- Atmosphere Build 0.6 native UI with cinematic fullscreen layout, evidence-board scene art, custom meters, buttons, case tabs and scene status strip
- Native local save data in `%LOCALAPPDATA%\\Lumorix\\EchoProtocol\\saves`
