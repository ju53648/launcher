# Quick Start - Graveyard Shift

1. Open PowerShell in this game folder.
2. Run: ./scripts/start-game.ps1
3. Validate parser: ./scripts/validate-runtime.ps1

Notes:
- Native runtime host ships as runtime/GraveyardShift.exe
- Legacy bootstrap remains available as runtime/Launch-GraveyardShift.ps1
- Uses modular bootstrap from src/modules
