# Echo Protocol Cutscene Asset Slots

Diese Slots sind jetzt direkt im Spiel verdrahtet. Wenn spaeter echte Bilder oder Videos in den Ordner
`public/assets/echo-protocol/cutscenes/slots/`
gelegt werden, zieht das Cutscene-Overlay automatisch die erste passende Datei.

## So funktioniert es

- Bilder und Videos koennen einfach in den Slot-Ordner gelegt werden.
- Akzeptierte Formate pro Slot: `.mp4`, `.webm`, `.png`, `.jpg`, `.jpeg`
- Das Spiel prueft die Dateinamen in genau dieser Reihenfolge.
- Wenn noch nichts vorhanden ist, erscheint automatisch der eingebaute Placeholder.
